**Web Application** එකක් යනු හුදෙක් තොරතුරු පෙන්වීමට එහා ගිය, පරිශීලකයා සමඟ ගනුදෙනු කරන (**Interactive**) සහ සංකීර්ණ කාර්යයන් කළ හැකි මෘදුකාංගයකි. මෙහිදී **Database** සහ **Authentication** (Login) වැනි දෑ අත්‍යවශ්‍ය වේ.

- **Analogy:** අන්තර්ජාලය හරහා ක්‍රියා කරන පරිගණක මෘදුකාංගයක් වැනිය.

- **Key Feature:** මෙය **Dynamic** වේ. දත්ත ඇතුළත් කිරීම, සැකසීම (Processing) සහ ගබඩා කිරීම සිදු කරයි.

- **Example:** Facebook, Gmail, Online Banking පද්ධති.

#### [[Web Site]] සහ [[Web Application]] අතර සම්බන්ධය (The Gray Area)

මෙය තමයි වැඩිපුරම Overlap වන තැන.

- සෑම **Web Application** එකක්ම තාක්ෂණිකව බලන විට **Web Site** එකක් ද වේ (මන්ද එය URL එකක් හරහා Browser එකකින් බලන බැවිනි).

- නමුත් සෑම Web Site එකක්ම Web Application එකක් නොවේ.

- **උදාහරණය:** ඔබ පුවත්පත් වෙබ් අඩවියකට ගොස් ප්‍රවෘත්ති කියවනවා නම් එය **Web Site** එකකි. නමුත් ඔබ එහිම ඇති "Search" bar එකේ යමක් සොයන විට හෝ Login වී Comment එකක් දමන විට, එය **Web Application** එකක ලක්ෂණ පෙන්වයි. අද කාලයේ බොහෝ වෙබ් අඩවි මේ දෙකේම එකතුවක් ලෙස ක්‍රියා කරයි.

**සම්බන්ධය**

- **[[Web Page|Web Pages]]** කිහිපයක් එකතු වූ විට **Web Site** එකක් ලැබේ.
- **Web Site** එකකට **Logic** සහ **Database** එකතු කළ විට එය **Web Application** එකක් වේ.
- **Web Application** එකක වේගය වැඩි කිරීමට සහ Reload වීම නැවැත්වීමට එය **[[SPA (Single Page Application)|SPA]]** ක්‍රමයට නිර්මාණය කරයි.
