
**SPA** යනු Web Application එකක් නිර්මාණය කරන නවීන ක්‍රමයකි. මෙහි ඇති විශේෂත්වය වන්නේ මුළු Application එකම **එකම Web Page එකක්** (index.html) තුළ ක්‍රියා කිරීමයි.

ඔබ යම් තැනක් Click කළ විට සම්පූර්ණ පිටුව **Reload** වන්නේ නැත. ඒ වෙනුවට වෙනස් විය යුතු දත්ත කොටස පමණක් **JavaScript** හරහා යාවත්කාලීන වේ.

**SPA** යනු වෙබ් අඩවියක් භාවිතා කරන අතරතුර එය නැවත නැවත **Reload** (පිටුව සම්පූර්ණයෙන්ම අලුත් වීම) නොවී, අවශ්‍ය දත්ත පමණක් යාවත්කාලීන කරමින් පරිශීලකයාට ඉතා වේගවත් අත්දැකීමක් ලබා දෙන **Web Application** මාදිලියකි.

#### SPA එකක ඇති "Single Page" එක කුමක්ද?

සාමාන්‍ය වෙබ් අඩවියක (MPA) පිටු 10ක් තිබේ නම්, එහි HTML ගොනු 10ක් පවතී. නමුත් SPA එකක ඇත්තේ **එකම එක HTML ගොනුවක්** පමණි (සාමාන්‍යයෙන් එය `index.html` වේ).

- **එය වැඩ කරන්නේ මෙහෙමයි:** මුලින්ම වෙබ් අඩවියට යන විට එම තනි HTML පිටුව සහ එයට අදාළ **JavaScript** එකවර Load වේ. ඉන්පසු ඔබ වෙනත් පිටුවකට (උදා: Profile page) යන විට, ඇත්තටම අලුත් පිටුවක් Load වන්නේ නැත. JavaScript මගින් දැනට ඇති පිටුවේ පෙනුම වෙනස් කර, අලුත් දත්ත එහි "පුරවනු" ලබයි.

#### SPA ක්‍රියා කරන ආකාරය (The Mechanism)

SPA එකක පිටු Reload නොවී ක්‍රියා කිරීමට ප්‍රධාන තාක්ෂණික කරුණු 3ක් බලපායි:

1. **Client-side Routing:** සාමාන්‍යයෙන් URL එකක් වෙනස් වන විට Browser එක පිටුවක් සොයයි. නමුත් SPA වලදී URL එක වෙනස් වුවත්, JavaScript විසින් එය හඳුනාගෙන පිටුව Reload වීමට ඉඩ නොදී අදාළ **Component** එක තිරයට පෙන්වයි.
2. **[[AJAX]] / Fetch API:** පිටුව Reload නොවී පසුබිමෙන් (Background) **[[Server]]** එක සමඟ කතා කර දත්ත (Data) ලබා ගැනීමට මෙය භාවිතා කරයි. 
3. **[[DOM]] Manipulation:** ලබාගත් දත්ත භාවිතා කර, සම්පූර්ණ පිටුවම අඳින්නේ නැතිව, වෙනස් විය යුතු කුඩා කොටස පමණක් JavaScript මගින් වෙනස් කරයි.

#### වාසි සහ අවාසි (Pros & Cons)

**වාසි:**

- **User Experience (UX):** මොබයිල් ඇප් එකක් භාවිතා කරනවා වැනි ඉතා සුමට අත්දැකීමක් ලැබේ.
- **Less Bandwidth:** සෑම විටම සම්පූර්ණ HTML එකම Download කිරීමට අවශ්‍ය නොවන නිසා දත්ත පිරිවැය අඩුය.
- **Caching:** අවශ්‍ය කරන Static ගොනු එකවර ගබඩා කරගන්නා නිසා පසුව ඉතා වේගයෙන් වැඩ කරයි.

**අවාසි:**

- **Initial Load Time:** මුල් වරට ඇතුළු වන විට සියලුම JavaScript ගොනු Load වීමට මද වෙලාවක් ගත විය හැක.
- **SEO Issues:** සෙවුම් යන්ත්‍ර (Google) වලට මුලින්ම හමුවන්නේ හිස් HTML පිටුවක් නිසා වෙබ් අඩවිය Rank කිරීමට තරමක් අපහසු විය හැක (නමුත් දැන් මෙයට විසඳුම් ඇත).

#### සැබෑ ලෝකයේ උදාහරණ

- **Gmail:** ඔබ Email එකක් Click කළ විට පිටුව Reload නොවී පණිවිඩය දර්ශනය වේ.
- **Facebook / Instagram:** ඔබ Scroll කරන විට හෝ Comment කරන විට පිටුව Reload වන්නේ නැත.
- **Google Maps:** සිතියම එහා මෙහා කරන විට මුළු පිටුවම Reload නොවී අලුත් සිතියම් කොටස් දිස්වේ.

**සාරාංශය:** SPA යනු එකම HTML පිටුවක් ඇතුළත, JavaScript ආධාරයෙන් Content පමණක් වෙනස් කරමින්, පරිශීලකයාට Reload රහිත අත්දැකීමක් ලබා දෙන ක්‍රමයකි.
