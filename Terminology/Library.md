Library එකක් කියන්නේ අපිට අවශ්‍ය වෙලාවට අපේ ප්‍රයෝජනයට ගන්න පුළුවන්, වෙනත් අය විසින් කලින් ලියපු කේත (Code) එකතුවක්.

හිතන්න ඔබ නිවසක් සාදනවා කියා. ඔබට ජනෙල් සහ දොරවල් අවශ්‍යයි.

- **Library එකක් නොමැතිව:** ඔබ විසින්ම ලී කපා, මැන, ජනෙල් සහ දොරවල් මුල සිටම සෑදිය යුතුය. මෙයට බොහෝ කාලයක් ගතවේ.
- **Library එකක් ඇතිව:** ඔබ සූදානම් කළ ජනෙල් සහ දොරවල් විකුණන කඩයකට (Library) ගොස්, ඔබට අවශ්‍ය ප්‍රමාණයන් තෝරාගෙන ඒවා කෙලින්ම ඔබේ නිවසට සවි කරයි.

මෙහිදී එම "කඩය" යනු **Library** එකයි. එහි ඇති "ජනෙල් සහ දොරවල්" යනු **Pre-written code** ය.

ඔබේ Program එක ඇතුළේදී යම් විශේෂිත Library එකක් භාවිතා කිරීමට නම්, මුලින්ම එය **Import** හෝ **Include** කරගත යුතුය.

උදාහරණ කිහිපයක්:

- **Python:** ඔබට දත්ත විශ්ලේෂණය කිරීමට අවශ්‍ය නම් **Pandas** හෝ **NumPy** වැනි Library භාවිතා කළ හැක.
- **C:** ඔබ නිතරම භාවිතා කරන `#include <stdio.h>` යනු Input/Output සඳහා වන Library එකකි.
- **Web Development:** UI එකක් ඉක්මනින් සාදා ගැනීමට **React** වැනි Library භාවිතා කරයි.

**Reinventing the wheel:** රෝදය නැවත සොයා ගැනීමට අවශ්‍ය නැත. උදාහරණයක් ලෙස, සංකීර්ණ ගණිතමය ගණනය කිරීමක් කිරීමට ඔබම formula එකක් ලියනවා වෙනුවට, ඒ සඳහා ඇති **Math Library** එකක් භාවිතා කළ හැක.

**[[Abstraction]]:** Library එක ඇතුළේ code එක වැඩ කරන සංකීර්ණ ආකාරය අපට දැනගැනීමට අවශ්‍ය නැත. අප කළ යුත්තේ එය භාවිතා කිරීම පමණි.

**Tested:** මේවා බොහෝ පිරිසක් භාවිතා කර ඇති නිසා වැරදි (**Bugs**) අඩුය.

**Control:** මෙතනදී Application එකේ පාලනය තියෙන්නේ සම්පූර්ණයෙන්ම ඔයා අතේ (Developer අතේ). ඔයාට අවශ්‍ය වෙලාවට, අවශ්‍ය තැනදී Library එකේ තියෙන Code එක Call කරන්න පුළුවන්.

- **[[Module]]:** Library එකක් ඇතුළේ ඇති කුඩා කොටස් වේ. (උදා: ගණිතමය වැඩ සඳහා ඇති Library එකක `Trigonometry` යනු එක් Module එකක් විය හැක).

- **[[API]] (Application Programming Interface):** Library එකක ඇති පහසුකම් අපේ Program එකට ලබා ගැනීමට භාවිතා කරන "දොරටුව" හෝ ක්‍රමයයි.

- **[[Dependency]]:** ඔබේ Project එක නිවැරදිව වැඩ කිරීමට වෙනත් Library එකක් මත යැපෙන්නේ නම්, එය Dependency එකක් ලෙස හඳුන්වයි.

- **[[Package Manager]]:** Library සොයා ගැනීමට, Install කිරීමට සහ Update කිරීමට භාවිතා කරන මෘදුකාංගයකි.
	උදාහරණ: **npm** (JavaScript), **pip** (Python), **pacman/yay** (Arch Linux).
